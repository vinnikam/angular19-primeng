ANGULAR.
Angular es un framework de desarrollo web creado por Google que permite construir aplicaciones web dinámicas,
interactivas y robustas. Está basado en TypeScript, un superconjunto de JavaScript,
y sigue una arquitectura basada en componentes, lo que hace que el desarrollo sea modular y reutilizable.

Características Principales de Angular
Arquitectura Basada en Componentes: Cada parte de la interfaz de usuario (UI) se divide en componentes individuales,
lo que facilita su mantenimiento y escalabilidad.

Two-Way Data Binding: Sincroniza automáticamente los datos entre el modelo y la vista,
lo que reduce la cantidad de código necesario para manejar interacciones.

Inyección de Dependencias: Proporciona una forma estructurada y eficiente de gestionar servicios y
otros recursos compartidos entre componentes.

Rendimiento Alto: Utiliza herramientas como el compilador Ahead-of-Time (AOT) y
la renderización Reactiva para optimizar el rendimiento.

Soporte para PWA (Progressive Web Apps): Permite crear aplicaciones web que funcionan como apps nativas en móviles
y computadoras.

Router Incorporado: Gestiona la navegación entre diferentes vistas de la aplicación,
facilitando la creación de aplicaciones de una sola página (Single Page Applications - SPAs).

2. Si no se encuentra instalar.

    https://nodejs.org

    Nuevamente validar

3. Verificar si se tiene instalado

    ng version

4. Instalar el angular última versión

    npm install -g @angular/cli

5. Crear proyecto angular

    ng new nombre-de-tu-proyecto
    cd nombre-de-tu-proyecto
    ng serve
